# puphpet-eelly
Vagrant development environment for eelly phper, built using PuPHPet

## info

* centos7
* php
  - bcmath
  - cli
  - intl
  - xml
  - mbstring
  - mcrypt
  - mysqlnd
  - igbinary
  - amqp
  - memcached
  - mongodb
  - redis
  - phalcon

## command
```
# 启动
vagrant up
# 配置更新
vagrant provision
# 重启
vagrant reload
# 挂起
vagrant halt 
```
