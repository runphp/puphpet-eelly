# puphpet-eelly
Vagrant development environment for eelly phper, built using PuPHPet

## info

* centos7
* php5.5
* php-ext-memcached2.2.0
* php-ext-mongodb1.2.2
* php-ext-mongo1.6.14
* php-ext-phalcon2.0.13
* php-ext-redis3.1.0

## command
```
# 启动
vagrant up
# 配置更新
vagrant provision
# 重启
vagrant reload
# 挂起
vagrant halt 
```
