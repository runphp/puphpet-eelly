# puphpet-eelly
Vagrant development environment for eelly phper, built using PuPHPet

## info

* centos7
* php7.0.x
* php-ext-memcached3.0.3
* php-ext-mongodb1.2.6
* php-ext-phalcon3.0.4
* php-ext-redis3.1.1
* php-ext-swoole2.0.7

## command
```
# 启动
vagrant up
# 配置更新
vagrant provision
# 重启
vagrant reload
# 挂起
vagrant halt 
```
